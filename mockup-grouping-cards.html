Packing Groups — Card Layout Drafts

Four visual directions. Each uses the new grouping logic (time of day → weather → event → dress code). Pick one to build, or cherry-pick elements across them.

Proposed grouping logic

  1. Time of day — splits first. Buckets: morning (5a–11a), afternoon (11a–4p), evening (4p–9p), night (9p–5a). Derived from start_time, falls back to time_slot.
  2. Weather — splits next. Buckets: hot/sunny, warm/sunny, mild, cold, rainy, snowy. Per-date, per-time-of-day if forecast supports it.
  3. Event compatibility — swimwear, formal, and active dress codes never cross-group with others, even if other factors match.
  4. Dress code — the final merge filter. Same code = same group.

Grouping key = date + time_of_day + weather_bucket + dress_code. One key = one card.
