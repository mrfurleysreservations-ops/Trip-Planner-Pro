Proposed Standard: Trip Tab Layout

Every tab inside a trip (Itinerary, Expenses, Packing, Notes, Meals, Group) follows this exact skeleton. New tabs added later must conform.
1. Sticky header
position: sticky; top: 0 — back button · page title · up to 2 action buttons (Export, Import, Add, Edit…). Height ≈ 44px.
2. Sticky pill toggle
Primary view switcher, always 2 options, centered, same pill component as today's Calendar/List. Required on every tab.
3. Sticky chip row (optional)
Horizontally scrollable secondary filter (day chips, category chips, person chips). Tab-specific. Still inside sticky region when present.
4. Scrollable body
flex: 1; overflow-y: auto — all content below the sticky region scrolls. Bottom padding 72px so last item clears the sub-nav.
5. Fixed bottom sub-nav
Unchanged. 54px, 6 tabs, position: fixed; bottom: 0.
6. Optional FAB
Primary create action, bottom-right, bottom: 72px to clear sub-nav. Only on tabs with a clear "add one" primary action.
